# Smart Budget Manager 🏦💡

An AI-powered Android app that helps users manage their budget intelligently using **Gemini 2.0** and Google AI tools.

## Features
- ✅ AI-driven expense tracking & insights
- ✅ Smart budget recommendations
- ✅ Integration with **Vertex AI**
- ✅ User-friendly and intuitive UI

## Tech Stack
- **Kotlin**
- **Google Gemini 2.0**, **Vertex AI**
- **Android SDK**, **Firebase**
